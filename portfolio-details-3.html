Klasifikasi Biji Kopi Robusta yang telah Disangrai

Project information

  • Category: Data Science
  • Project URL: www.github.com/fikranltd/klasifikasi-biji-kopi-robusta

Tujuan dari penelitian ini yaitu untuk membangun sistem kecerdasan buatan yang dapat melakukan klasifikasi biji kopi robusta setelah proses roasting menggunakkan algoritma Convolutional Neural Network (CNN). Sebanyak 450 data dari 3 kategori hasil roasting ditraining dan testing. Data ditraining menggunakan 3 skenario dengan epoch, batch size, dan learning rate yang berbeda. Hasil ketiga skenario, didapati bahwa skenario pertama dengan epoch 200, batch size 16, dan learning rate 0,0001 mendapat akurasi tertinggi yaitu 95%. Scenario kedua dengan epoch 150, batch size 30, dan learning rate 0,0001 mendapat akurasi terendah yaitu 90%. Sedangkan scenario ketiga dengan epoch 100, batch size 24 dan learning rate 0,001 mendapat akurasi menengah yaitu 95%.
